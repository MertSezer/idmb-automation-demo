Known Limitations

Important project boundaries to present honestly during the demo.

  • Profile-name validation is the stable core capability of the system.
  • Character extraction is still experimental and may intentionally remain blank instead of returning noisy cross-row data.
  • Live IMDb access can be affected by anti-bot behavior, so replay-first presentation remains the safest online-demo strategy.
  • Screenshot evidence is the strongest differentiator of this project and should be highlighted in the walkthrough.
